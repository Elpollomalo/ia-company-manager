# 🤖 ia-company-manager

Sistema multi-agente genérico y agnóstico para la gestión de proyectos a través de un pipeline de refinamiento de datos por etapas.

- Operando localmente en Linux Mint mediante Claude Code.
- Estructura desacoplada: Los agentes procesan procesos puros; el contexto de cada proyecto se inyecta dinámicamente desde la carpeta 'vault/sources/'.
